This project uses both the Enterprise Library (DAAB) and DotNetZip.

Download the Enterprise Library Data Access Application Block using NuGet in Visual Studio.

Download DotNetZip from here: http://dotnetzip.codeplex.com/ and follow the installation instruction.

See the DAAB.vb form for examples of how to use the DAAB.

See the app.config file to see what needs to be added to a new project.
